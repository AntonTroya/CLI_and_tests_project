В первом проекте три файла:

- CLI.py
- file_operations.py (операции над файлами: копирование, удаление)
- tests_file_operations.py (тестирование копирования и удаления файлов)


					Запуск через CMD

Необходимо перейти в директорию проекта, в нем должны быть файлы: CLI.py, file_operations.py, tests_file_operations.py



Команды для работы

- копирование файла:
python CLI.py copy [путь_к_исходному_файлу] [путь_назначения]

- удаление файла или папки:
python CLI.py delete [путь_к_файлу_или_папке]



Запуск тестов:
python tests_file_operations.py
